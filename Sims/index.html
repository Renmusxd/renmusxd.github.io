Emergent Behavior

Birds on a Torus

This is a little simulation of flocking behaviour on a torus. Each of the creatures below follows a set of rules:

  • Periodically generate a random reference velocity vector \(\vec{a}\)
  • Let \(\vec{b}\) be the average velocity of k nearest neighbors (within search distance MaxD)
  • Set own velocity to \(\alpha * \vec{a} + (1-\alpha)*\vec{b}\)

Alpha: 0

K: 1

MaxD: 100
